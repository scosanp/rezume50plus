# Резюме 50+

Статический сайт-витрина для сервиса подготовки резюме опытным специалистам 50+. Сайт сделан по ТЗ как лидогенерационный прототип без CMS и без упоминания автоматизации/ИИ.

## Локальный запуск

```bash
node server.mjs
```

Откройте `http://localhost:4174`.

## Публикация

Проект рассчитан на GitHub Pages из ветки `main`, папка `/`.

Ожидаемый адрес:

```text
https://scosanp.github.io/rezume50plus/
```

## Что внутри

- Главная страница с Hero, социальным доказательством, отзывами и CTA.
- Страницы услуг, процесса, эксперта, отзывов/кейсов, блога, контактов и политики.
- Адаптивная верстка, бургер-меню, фиксированные кнопки звонка и мессенджеров.
- Формы заявки с сохранением лидов в `localStorage` для прототипа.
- SEO: `robots.txt`, `sitemap.xml`, мета-теги, Schema.org.
- Сгенерированный временный портрет эксперта: `assets/expert-portrait.jpg`.
